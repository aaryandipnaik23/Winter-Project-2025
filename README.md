# Winter-Project-2024

## Project Title
Student Management System

---

## Description
This project is a Java-based Student Management System developed as part of my Winter Project 2024.
It allows users to manage student records efficiently using core Java concepts and data structures.

---

## Features
- Add new student
- View all students
- Search student by ID
- Delete student record
- Menu-driven interface
- Uses OOP principles

---

## Tech Stack
- Java
- Data Structures (ArrayList)
- OOP Concepts

---

## How to Run
1. Clone the repository  
2. Compile all files inside `src` folder  
3. Run `Main.java`
